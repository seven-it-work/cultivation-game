# cultivation-game



# 战斗

每次攻击消耗精力2点，当精力==0时，强制休息，休息后恢复 恢复力值的精力+生命。

所以的属性值（除去恢复力）=精力/总精力*属性值

精力低于30%才能主动休息

# 修炼

世界的总灵力值只够，当前世界5人修炼到顶峰

当人被击杀，灵力会回归被击杀地的自然，待自然按各自的灵力恢复 来生成灵力
